Try76 Diagram Collection

Distribution-first path-loss experts: a simpler two-stage model than the older prior-heavy systems, with separate diagrams for the model, height conditioning, expert routing, dataset splitting, and how the output distribution was chosen.

1. Try76 Full System

High-level view of the two-stage path-loss expert: minimal spatial inputs, topology and LoS/NLoS routing, a height-conditioned distribution head, then a map decoder trained on valid ground pixels only.

2. Sinusoidal Height And Variance

Shows the height path explicitly: the scalar UAV height becomes a log-spaced sinusoidal embedding, modulates the encoder through FiLM, changes the predicted mixture means and widths, and then propagates that drift into the decoder. This is the part that explains how height-dependent variance enters the model.

4. How The Distribution Is Chosen

This diagram ties the docs to the model choice. Try76 looks at histogram-fit rankings and uses a multi-component GMM for path loss; Try77 keeps the same philosophy but adapts it to spike-plus-tail spread distributions.
